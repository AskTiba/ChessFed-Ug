## 🚀 Roadmap & Milestones

### Phase 1: Foundation & Awareness (COMPLETED ✅)
- [x] Public Awareness Hub (Calendar, Detail Pages).
- [x] National Rankings & Grand Prix Engine.
- [x] National Team Selection Hub & Year Filtering.

### Phase 2: Private Member & Club Portal (IN PROGRESS ⏳)
- [x] Player Dashboard & Profile Linking.
- [x] Profile Editor (Club/Region/Bio).
- [ ] **Club Management Console:** For Owners/Captains to manage 12-man rosters.
- [ ] **Roster Lock System:** Securely finalizing team lineups for the League.

### Phase 3: Federation Administration (Week 3)
- [x] Admin Dashboard (Oversight).
- [x] Tournament Initialization Form.
- [x] Grand Prix Point Distribution Tool.
- [ ] **News & Announcements:** Official federation press releases and event updates.

### Phase 4: Live Ops & Real-Time (Week 4)
- [x] Live Center UI (Pairings & Standings).
- [ ] Socket.io Integration for real-time broadcasts.
- [ ] Chess.js PGN Viewer integration.
